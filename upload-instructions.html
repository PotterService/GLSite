How to Upload the HTML Files

These instructions assume Gamers Link is using GoDaddy cPanel web hosting with File Manager.

GoDaddy cPanel Upload HelpDownload Files

Basic Upload Steps

Download the ZIP

Go to the Download page and download the demo website ZIP file.

Extract the ZIP

Right-click the ZIP file on your computer and extract/unzip it so you can see the HTML files and folders.

Log in to GoDaddy

Open the GoDaddy account that manages the web hosting for the website.

Open cPanel File Manager

Go to the hosting dashboard, open cPanel, and then open File Manager.

Open public_html

The public_html folder is usually where website files go. If there is an existing website, make a backup before replacing anything.

Upload the Files

Upload the extracted files and folders into public_html. Make sure index.html is in the main root folder.

Test the Website

Visit the website in a browser and check every page, image, button, and link.

Important Backup Reminder

Before replacing an existing website, download a backup copy of the current files. This makes it easier to restore the old site if needed.
