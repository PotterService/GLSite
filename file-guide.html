Demo Website File Guide

This page explains what each main file or folder is used for in the demo website.

Main Files

index.html — Home page / main landing pageabout.html — About or story pageboard-games.html — Board games pagecards.html — Trading cards pagedice.html — Dice and RPG accessories pagegallery.html — Photo gallery pagevisit.html — Contact, hours, address, and visit informationterms.html — Disclaimer, terms, takedown request, and ownership informationstyle.css — Website colors, layout, fonts, spacing, buttons, and page stylingscript.js — Interactive features such as popups, photo zoom, buttons, and mini featuresassets/ — Images, logos, and media used by the website

Optional Future Files

sitemap.xml — Helps search engines find website pagesrobots.txt — Tells search engines what they can crawl404.html — Custom missing-page screenfavicon files — Website icon files for browser tabs and phones
